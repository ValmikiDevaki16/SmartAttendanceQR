# 📱 SmartAttendanceQR

A QR Code Based Attendance Management System developed using **Kotlin** and **Firebase** for Android.

SmartAttendanceQR simplifies classroom attendance by allowing teachers to generate time-limited QR codes and students to mark attendance by scanning them. The system provides real-time attendance tracking, class management, and attendance analytics.

## ✨ Features

### 👨‍🏫 Teacher Module
- Teacher Registration & Login
- Create and Manage Classes
- Generate Unique Class Codes
- Start Attendance Sessions
- Generate Dynamic QR Codes
- Share QR Codes with Students
- Real-Time Attendance Tracking
- View Attendance Reports
- Delete Classes
- End Active Sessions

### 👨‍🎓 Student Module
- Student Registration & Login
- Join Classes Using Class Codes
- Scan QR Codes to Mark Attendance
- View Subject-wise Attendance
- Track Attendance Percentage
- Attendance History Dashboard

## 🛠️ Tech Stack

- **Language:** Kotlin
- **IDE:** Android Studio
- **Backend:** Firebase Authentication
- **Database:** Firebase Firestore
- **QR Generation:** ZXing Library
- **Architecture:** Activity-Based Android Application

## 🚀 Workflow

### Teacher Flow

1. Register/Login
2. Create Class
3. Generate Class Code
4. Start Attendance Session
5. Generate QR Code
6. Students Scan QR
7. Attendance Recorded
8. View Attendance Reports

### Student Flow

1. Register/Login
2. Join Class Using Code
3. Scan QR Code
4. Attendance Marked
5. View Attendance Statistics
